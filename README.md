EffectKit
=======

iOS / OS X framework that attempts to mimic Photoshop's Blending Options. As much as possible. Open .PSD file created in Photoshop and use assets directly in code.

Effects
=======

- Blur
- Stroke (outside, inside)
- Color overlay
- Gradient overlay
- Drop shadow
- Inner glow
- Outer glow
